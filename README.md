## How to Run This Bot
### Prerequisites
 - Python 3.7
 - wikipedia library
 
### Setup
- Clone this repo from github

- Get a BOT ID from the botfather.

- Create a secret_settings.py file:
     ```
     BOT_TOKEN = "your-bot-token-here"
     ```
### Run   
To run the bot use:
   ```
   python bot.py
   ```
### Notes
- You get a large amount of data on the first run
